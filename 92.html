Explicación:
La instancia de operador prueba si un objeto tiene en su cadena de prototipos la propiedad de prototipo de un constructor.
La instancia del operador prueba la presencia de constructor.prototype en la cadena de prototipos de objetos.
Ejemplo:
// definiendo constructores
función C () {}
función D () {}
var o = nuevo C ();
// verdadero, porque: Object.getPrototypeOf (o) === C.prototype
o instancia de C;
// falso, porque D.prototype no está en ninguna parte de la cadena de prototipos de o
o instancia de D;

Explanation:
The instance of operator tests whether an object has in its prototype chain the prototype property of a constructor.
The instance of operator tests presence of constructor.prototype in object prototype chain.
Example:
// defining constructors
function C(){}
function D(){}

var o = new C();
// true, because: Object.getPrototypeOf(o) === C.prototype
o instanceof C;
// false, because D.prototype is nowhere in o's prototype chain
o instanceof D;
